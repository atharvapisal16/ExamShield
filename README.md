# 🛡️ ExamShield: AI-Powered Student Survival Toolkit

**ExamShield** is an all-in-one exam preparation suite designed to turn pre-exam panic into productive focus. Built for students who need quick results, it combines AI logic with essential productivity tools in a sleek, dark-themed interface.

---

## 🚀 Key Features

* **🤖 Multi-AI Doubt Solver:** Get instant explanations for complex topics using integrated AI models.
* **⏱️ Pomodoro Study Timer:** Stay focused with customizable study intervals and break reminders.
* **📝 Quick Notes Generator:** Summarize long chapters into digestible bullet points quickly.
* **🗂️ Revision Flashcards:** Automated flashcard generation for active recall.
* **📁 PYQ Practice Bank:** Organize and access Previous Year Questions in one place.
* **🧘 Calm Mode:** An anxiety-relief interface to help you destress before the big day.

---

## 🛠️ Project Structure

The project is built using pure **HTML5, CSS3, and JavaScript**, making it lightning-fast and easy to deploy.

* `index.html` - The main dashboard/landing page.
* `doubt-solver.html` - AI-powered Q&A interface.
* `timer.html` - Focus and Pomodoro tool.
* `notes.html` - Note-taking and summarization module.
* `anxiety.html` - Calm mode for mental wellness.
* `js/` & `css/` - Contains all core logic and styling.

## 📦 How to Use Locally

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/atharvapisal16/ExamShield.git](https://github.com/atharvapisal16/ExamShield.git)
    ```
2.  **Open the Project**
    Simply open `index.html` in any modern web browser.
3.  **Configure AI**
    The app uses client-side API calls. You can input your own API keys (OpenAI/Gemini) within the application settings to enable the AI features.
